## Show version of projects inside container
```bash
tnp versions
```


## Firedev extension for Visual Studio Code 

Install vscode plugin

```bash
tnp vscode:global
```


## Dowload video as mp3

Install vscode plugin

```bash
tnp mp3 <url to video>
```
